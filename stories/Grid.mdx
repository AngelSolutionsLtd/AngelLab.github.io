## Auto Columns

<div className="section d-block py-3">
  <div className="row no-gutters">
    <div className="col-md-12">
      <div className="bd-example">
        <div className="container">
          <div className="row">
            <div className="col-sm-4 col-md-2">Col</div>
            <div className="col">Auto Column</div>
            <div className="col-sm-4 col-md-2">Col</div>
          </div>
        </div>
      </div>

      ```jsx
      <div class="row">
        <div class="col-sm-4 col-md-2">
          Col
        </div>
        <div class="col">
          Auto Column
        </div>
        <div class="col-sm-4 col-md-2">
          Col
        </div>
      </div>
      ```
      
      Auto-layout for flexbox grid columns also means you can set the width of one column and have the sibling
      columns automatically resize around it. You may use predefined grid classNamees (as shown above), grid mixins,
      or inline widths. Note that the other columns will resize no matter the width of the center column.
    </div>
  </div>
</div>

## Utilities


<div className="section d-block py-3">
    <div className="row">
        <div className="col-md-12">
            Change the value of the display property with our responsive display utility classes. We purposely support only a subset of all possible values for display. Classes can be combined for various effects as you need.

            Display utility classes that apply to all breakpoints, from <code className="highlighter-rouge">xs</code> to <code className="highlighter-rouge">xl</code>, have no breakpoint abbreviation in them. This is because those classes are applied from <code className="highlighter-rouge">min-width: 0;</code> and up, and thus are not bound by a media query. The remaining breakpoints, however, do include a breakpoint abbreviation.

            As such, the classes are named using the format:

            <ul>
                <li><code className="highlighter-rouge">.d-{'{'}value{'}'}</code> for <code className="highlighter-rouge">xs</code></li>
                <li><code className="highlighter-rouge">.d-{'{'}breakpoint{'}'}-{'{'}value{'}'}</code> for <code className="highlighter-rouge">sm</code>, <code className="highlighter-rouge">md</code>, <code className="highlighter-rouge">lg</code>, and <code className="highlighter-rouge">xl</code>.</li>
              </ul>
              

              Where <em>value</em> is one of:

              <ul>
                <li><code className="highlighter-rouge">none</code></li>
                <li><code className="highlighter-rouge">inline</code></li>
                <li><code className="highlighter-rouge">inline-block</code></li>
                <li><code className="highlighter-rouge">block</code></li>
                <li><code className="highlighter-rouge">table</code></li>
                <li><code className="highlighter-rouge">table-cell</code></li>
                <li><code className="highlighter-rouge">table-row</code></li>
                <li><code className="highlighter-rouge">flex</code></li>
                <li><code className="highlighter-rouge">inline-flex</code></li>
              </ul>

              <div className="bd-example">
                <div className="container">
                    <div className="row align-items-end">
                        <div className="col-4 d-none d-md-block">One of three columns</div>
                        <div className="col-4 d-lg-none">One of three columns</div>
                        <div className="col-4 d-none d-md-block">One of three columns</div>
                    </div>
                </div>
            </div>

            ```html
            <div class="container">
                <div class="row align-items-end">
                    <div class="col-4 d-none d-md-block">One of three columns</div>
                    <div class="col-4 d-lg-none">One of three columns</div>
                    <div class="col-4 d-none d-md-block">One of three columns</div>
                </div>
            </div>
            ```		

            The example above will toggle the display of the middle column until the <code class="highlighter-rouge">lg</code> breakpoint.														  
        </div>
    </div>
