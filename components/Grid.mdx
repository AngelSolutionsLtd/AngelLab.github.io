## Auto Columns

<div className="section py-3">
  <div className="row no-gutters">
    <div className="col-md-12">
      <div className="bd-example">
        <div className="container">
          <div className="row">
            <div className="col-sm-4 col-md-2">Col</div>
            <div className="col">Auto Column</div>
            <div className="col-sm-4 col-md-2">Col</div>
          </div>
        </div>
      </div>

      ```html
      <div class="row">
        <div class="col-sm-4 col-md-2">
          Col
        </div>
        <div class="col">
          Auto Column
        </div>
        <div class="col-sm-4 col-md-2">
          Col
        </div>
      </div>
      ```
      
      Auto-layout for flexbox grid columns also means you can set the width of one column and have the sibling
      columns automatically resize around it. You may use predefined grid classes (as shown above), grid mixins,
      or inline widths. Note that the other columns will resize no matter the width of the center column.
    </div>
  </div>
</div>
